## 🎓 Learning Path

Perfect for CKA/CKAD certification preparation:

1. **Basics** → Pods, Namespaces, kubectl
2. **Workloads** → Deployments, DaemonSets, StatefulSets
3. **Networking** → Services, Ingress, Network Policies
4. **Security** → RBAC, Pod Security Standards
5. **Storage** → Volumes, Persistent Volumes
6. **Scaling** → HPA, Resource Management
7. **Advanced** → Operators, Custom Resources
